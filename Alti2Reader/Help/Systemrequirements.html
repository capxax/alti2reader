System requirements

Getting Started ››

This program is developed with MS Visual C# using .NET Framework 4.0 Client package.

If this package is not installed setup program will advise You to download it from Microsoft site for free.

All other necessary DLLs components are installed with the program.


This program is tested on:

  • computer with Intel Pentium 4 2GHz CPU and 1GB RAM under MS Windows XP SP3.
  • computer with AMD Turion II Dual-core 2.5GHz CPU and 4GB RAM under MS Windows 7 x64 Home Edition SP1.
  • on computer with Intel Core i7 2.7GHz CPU and 4GB RAM under MS Windows 7 x64 Professional Edition SP1.


You should install Altimaster USB drivers to communicate with N3 or N3Audio devices.

Drivers can be download from here.


To communicate with Neptune 2 device You should use IrDA adapter.

This program is tested with Tekram IRMate-210 serial adapter under Windows 7 x64 Home Edition SP1 and with Tekram IRMate-410U USB adapter under Windows XP SP3. Under Windows XP program is more stable in case of more stable drivers. Drivers for IRMate-210 for Windows 7 x64 and instructions how to install and configure them You can download from here.
